# Example Project

A demonstration of the project documentation system.

## Overview

This project exists to show how projects are rendered on INHUMAN SYSTEMS. Each project lives in its own folder with a `README.md` file that gets parsed and displayed.

## Features

- **Markdown support** — headers, lists, code, links
- **Images** — place in `assets/` folder, reference as `assets/image.png`
- **Code blocks** — syntax highlighted automatically
