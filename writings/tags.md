## Tag Guidelines

- Use lowercase, single words when possible
- Prefer existing tags over creating new ones
- Each writing should have 3-5 tags
- Tags should be conceptual, not structural (e.g. `AI` not `essay`)
- Check this file before adding new tags to maintain consistency
